# Análise de Evasão de Clientes - Telecom X

## Descrição do Projeto
O objetivo foi realizar uma análise exploratória nos dados da empresa fictícia Telecom X para identificar os principais fatores que levam os clientes a cancelarem seus serviços.

## Objetivos da Análise

* Identificar o perfil dos clientes que cancelam o serviço.
* Entender quais fatores mais influenciam na decisão.
* Gerar insights para que a equipe de Data Science possa desenvolver modelos preditivos e para que a empresa possa criar estratégias de retenção.

## Ferramentas Utilizadas

* Linguagem: Python
* Bibliotecas: Pandas (para manipulação de dados), Matplotlib e Seaborn (para visualização de dados)
* Ambiente: Google Colab
  
## Principais Descobertas

* A taxa de evasão geral da empresa é de aproximadamente 27%.
* Clientes com contrato mensal têm uma propensão muito maior a cancelar.
* Clientes com faturamento mensal mais alto apresentam maior taxa de evasão.
  
## Como Utilizar?

* Clone o repositório.
* O arquivo Desafio.ipynb contém todo o código e a análise detalhada.
* Para executar, basta abrir o notebook em um ambiente como o Google Colab ou Jupyter Notebook.
